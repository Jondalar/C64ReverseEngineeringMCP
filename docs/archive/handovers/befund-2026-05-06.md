## Was implementiert ist

### Sprint 113 (chip-level 1:1 VICE port)
- **Alarm system** (Spec 149): `src/runtime/headless/alarm/alarm-context.ts` — 1:1 VICE alarm.c
- **CIA 6526** (Spec 145): `src/runtime/headless/cia/cia6526-vice.ts` + ciat.ts + cia-tod.ts + cia-sdr.ts — alarm-driven 1:1 VICE ciacore.c, IFR delay-line, write_offset, TOD, SDR
- **CPU 65xx** (Spec 146): `src/runtime/headless/cpu/cpu65xx-vice.ts` — shared core, alarm-aware, microcode-table flags, branch-delay flag-based, interrupt-cycle accurate, SO-pin input line
- **VIA 6522** (Spec 147): `src/runtime/headless/via/via6522-vice.ts` + via1d1541.ts + via2d1541.ts — alarm-driven 1:1 VICE viacore.c, full T1/T2/SR/ILA/ILB/CA-CB modes
- **VIC** (Spec 150): `src/runtime/headless/vic/vic-ii-vice.ts` — B-level (cycle-accurate bus stealing + raster IRQ + register R/W; no pixel rendering)
- **SID** (Spec 151): `src/runtime/headless/sid/sid.ts` — B-level (register state correct, no audio)
- **Reset state** (Spec 148): VICE RAMInit logic + per-chip reset audit
- **PLA banking** fix: `(~dir | data) & 7` mem_config formula + VICE data_read formula on $01 read
- **Drive track-count** fix: HeadPosition bumped from G64 parser

### Spec 152 (swimlane diff infrastructure)

### Spec 153 (drive GCR/motor/head)
- `src/runtime/headless/drive/gcr-shifter.ts` (392 LOC) — bit-stream rotation 1:1 VICE rotation.c, 4-zone density [32,30,28,26], MSB-first, byte-ready emission
- `src/runtime/headless/drive/sync-detector.ts` (168 LOC) — SYNC# pattern 1:1 VICE gcr.c (10-bit threshold, register `(reg<<1)&0x3fe | bit`)
- `src/runtime/headless/drive/via2-gcr-shifter-coupling.ts` (109 LOC) — VIA2 backend wrapping shifter
- `cpu65xx-vice.ts`: SO-pin input line + edge-detect (kept for future use)
- `drive-cpu.ts`: byte-ready callback sets `cpu.reg_p |= 0x40` directly (matches VICE drivecpu_set_overflow)
- Feature-flagged: `C64RE_USE_GCR_SHIFTER=1` enables (default off)
