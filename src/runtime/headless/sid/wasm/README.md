# reSID WASM module (committed prebuilt) — Spec 703

This directory holds the **prebuilt** reSID audio engine so a plain checkout /
`npm install` gets working SID audio with **no emscripten and no compile step**.

Files (generated by `npm run build:resid-wasm`, committed to git):

- `resid.mjs` — emscripten ES6 glue (MODULARIZE/EXPORT_ES6)
- `resid.wasm` — compiled reSID + shim (portable bytecode, all platforms)

`npm run build:mcp` copies these into `dist/runtime/headless/sid/wasm/` next to
the compiled engine (`resid-wasm-engine.js`), which loads `./wasm/resid.mjs`.

## Rebuilding (maintainer only — needs emscripten)

```bash
brew install emscripten      # or set up emsdk; provides emcc/em++
npm run build:resid-wasm     # compiles third_party/resid/*.cc + resid_shim.cc
git add src/runtime/headless/sid/wasm/resid.mjs src/runtime/headless/sid/wasm/resid.wasm
```
